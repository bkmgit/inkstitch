Платформа для разработки дизайнов вышивки с открытым исходным кодом основаная на Inkscape.

Ink/Stitch стремится быть полноценной, кроссплатформенной средой для создания цифровой вишивки, онснованой исключительно на бесплатном ПО с открытым исходным кодом. Наша цель быть доступными для энтузиастов и предоставлять широкие возможности для профессионалов машинной вышивки.
